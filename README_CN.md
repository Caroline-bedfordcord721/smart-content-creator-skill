# Smart Content Creator - Claude AI 智能内容创作技能

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Version](https://img.shields.io/badge/version-1.0.0-blue.svg)](https://github.com/ginobefun/smart-content-creator-skill)

将阅读材料和个人见解转化为精致、可分享的内容——从博客文章、社交媒体到可视化和图表。强调自然、类人类的表达，避免 AI 风格的写作模式。

[English](README.md)

## 🌟 核心特性

通过 **6 种核心内容类型 + 高级可视化** 转化你的学习成果：

### 文字格式
- 📝 **博客文章** - 引人入胜的长文内容（2000-4000 字）
- 📱 **小红书笔记** - 真实的社交媒体内容（中文平台，800-1500 字符）
- 🐦 **Twitter 长文** - X 平台的专业文章（1000-4000 字）

### 可视化格式
- 🎨 **思维导图** - 层次化的知识组织
- 📊 **信息图表** - 数据驱动的视觉故事讲述
- 📈 **图表和图示** - 流程图、概念图、时间线等

### 核心能力
- 💎 **深度内容创作** ⭐ 新功能 - 多层次思考、根本原因分析、反直觉洞察
- 🎭 **个人声音保留** ⭐ 新功能 - 捕捉并保持你的真实写作风格
- ✨ **自然表达** - 内容读起来像人类写作，而非 AI 输出
- 🎯 **灵活风格** - 多种写作方式，而非僵化模板
- 🌏 **文化意识** - 针对目标语言（英语/中文）进行本地化
- 🚫 **反 AI 模式** - 全面指南避免机械化写作
- 🎨 **智能可视化** - 根据信息结构选择合适的图表类型

## 🎯 技能功能

本技能连接阅读/理解与分享/创作之间的桥梁。它帮助你：

1. **转换阅读笔记** → 博客文章、Twitter 推文或社交媒体内容
2. **转化见解** → 真实、文化适宜的内容
3. **可视化知识** → 思维导图、信息图表、流程图、概念图
4. **保持真实声音** → 自然、类人类的内容
5. **适应平台** → 为每个媒介选择合适的语气和格式
6. **澄清复杂想法** → 增强理解的视觉表示

## 🚀 安装方法

### 前置要求

- Claude 桌面应用或 Claude 网页版（claude.ai）
- Claude Pro 订阅（推荐以获得最佳体验）

### 安装步骤

1. **下载** 技能文件：[smart-content-creator.skill](smart-content-creator.skill)

2. **导入到 Claude：**

   **桌面应用：**
   - 打开 Claude 桌面应用
   - 点击右上角头像
   - 选择 Settings → Skills
   - 点击 "Import Skill"
   - 选择下载的 `.skill` 文件

   **网页版：**
   - 访问 [claude.ai](https://claude.ai)
   - 点击右下角设置图标
   - 进入 "Skills" 标签页
   - 点击 "Import" 按钮
   - 上传 `.skill` 文件

3. **开始创作！** 当你请求内容创作时，技能会自动激活。

## 💎 深度内容与个人声音

### 什么让内容真正有价值？

两个关键维度区分了卓越内容与通用输出：

#### 1. 真正的深度 🎯

超越表面观察，达到改变读者思维方式的洞察。

**技能帮助你**：
- **多层次思考**：观察 → 模式 → 根本原因 → 影响 → 反直觉洞察
- **"为什么链"分析**：深挖 3-5 层以发现基本真理
- **跨领域连接**：以意想不到的方式链接不同领域的想法
- **挑战假设**：质疑"众所周知"的事情，找到真正正确的
- **探索张力**：拥抱复杂性而非过度简化

**深度递进示例**：
```
表面："远程工作变得流行"
模式："生产力因角色类型而异"
根本原因："协作频率而非地点预测了结果"
影响："工作设计比工作地点更重要"
反直觉："更多沟通工具降低了效率"
```

#### 2. 真实的个人声音 🎭

你独特的视角是让内容不可替代的关键。

**技能捕捉**：
- **你的思维模式**：你如何自然地处理和解释
- **你的视角**：你注意到而他人忽略的
- **你的表达风格**：明显属于你的节奏和语气
- **你的价值观**：你关心的自然流露
- **你的演变**：你的理解如何改变

**声音发现问题**：
- 什么让你感到惊讶？
- 你如何向朋友解释事情？
- 什么感觉自然 vs. 强迫？
- 你的真实立场是什么？

**结果**：听起来明显像你的内容，而非通用的 AI 输出。

### 工作原理

**在生成任何内容之前，技能会**：
1. 通过仔细观察理解你的真实声音
2. 评估主题所需的深度级别
3. 提出问题以发现你独特的洞察
4. 调整风格框架以匹配你的自然表达
5. 创建既有实质又真实属于你的内容

**这不是模板化——而是发现并放大你思维的独特之处。**

## 🎯 技能的独特之处

### 1. 自然、类人类的写作

**传统 AI 写作**：
```
此外，这种方法有几个好处。而且，它很容易实施。
另外，结果很显著。
```

**使用 Smart Content Creator**：
```
这种方法效果很好，实施起来也很简单。
结果不言自明。
```

### 2. 灵活风格，而非僵化模板

你获得的是可以混合和适应的**风格框架**，而非固定模板：
- 深思熟虑的探索者
- 实用指南
- 分析观察者
- 故事讲述者
- 对话专家

### 3. 文化本地化

**英文内容**：地道表达、自然节奏、文化相关性

**中文内容**：
- 真实的口语化表达
- 自然的中文思维模式
- 适当的网络用语
- 没有翻译痕迹

### 4. 全面的反 AI 指南

内置防护措施防止常见的 AI 写作模式：
- ❌ 过度的过渡短语
- ❌ 过度使用引号
- ❌ 流行语过载
- ❌ 公式化结构
- ❌ 泛泛建议

### 5. 智能可视化

- 根据信息结构选择合适的图表类型
- 多种输出格式（Mermaid、文本、描述）
- 清晰、有目的的设计

## 💡 技巧与最佳实践

### ✅ 推荐做法

1. **提供上下文**：分享你的理解，而非只是来源
2. **具体说明**："初学者的对话语气" > "做得好"
3. **请求可视化**：在解释关系时要求图表
4. **迭代**：如果第一版不完美，请求调整
5. **分享你的声音**：让你的个性影响内容
6. **检查真实性**：确保听起来像你，而非通用 AI

### ❌ 需要避免

1. 不提供上下文就期待完美
2. 请求"AI 防护"内容（所有内容都应听起来像人类）
3. 过度指定格式（让内容自然呼吸）
4. 不反馈就接受通用输出
5. 忘记审查个人声音

## ❓ 常见问题

**问：内容会听起来像 AI 生成的吗？**
答：不会。技能特别避免 AI 写作模式，专注于自然、类人类的表达。

**问：我能同时获得文字和可视化吗？**
答：当然！你可以请求带图表的博客文章、带图表的社交媒体等。

**问：提供哪些可视化格式？**
答：Mermaid 代码（可渲染）、基于文本的图示（ASCII 风格）和设计师的详细描述。

**问：支持中文内容吗？**
答：是的，特别优化了真实的中文表达，避免翻译腔语言。

**问：能从头创建内容吗？**
答：它最适合与源材料和你的洞察一起工作。提供清晰的上下文以获得最佳结果。

**问：思维导图和概念图有什么区别？**
答：思维导图是层次化的（树形结构）。概念图显示带有标记连接的网络关系。
